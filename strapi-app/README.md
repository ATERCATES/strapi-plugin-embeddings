# Dribo Content CMS

Content CMS for Dribo

## Development

```
docker-compose pull
docker-compose --env-file path/to/.env up
```

### Env example

```
DATABASE_CLIENT=postgres
DATABASE_NAME=strapi
DATABASE_HOST=postgres
DATABASE_PORT=5432
DATABASE_USERNAME=strapi
DATABASE_PASSWORD=strapi
ADMIN_JWT_SECRET=whatever
```
